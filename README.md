# Unit UI 前端框架

Unit UI前端框架是一款基于Bootsrtap 3.3.5的深度美化的前端框架，完全兼容于Bootstrap 3.3.5。本框架是一款完全开源的前端框架，使用者可以根据自身功能需求来更和使用此框架。

本框架在Bootsrtap 3.3.5原有的基础上，又新增了常用的css类，以及部分javascript插件和Admin管理面板。同时需要指出的是，第一，本框架不兼容IE8一下浏览器，推荐使用现代常用的浏览器。第二，本框架没有把所有常用的插件都封装好供开发者使用，只提供部分常用的插件。这样做的目的是因为术业有专攻，有大量的优秀js插件可以和本框架同时使用，这样可以供开发社更好的根据自身需求去定制开发网站。
